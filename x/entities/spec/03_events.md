# Events

| Type           | Attribute Key | Attribute Value |
|----------------|---------------|-----------------|
| create_project | tx_hash       | {tx_hash}       |
| create_project | sender_did    | {sender_did}    |
| create_project | project_did   | {project_did}   |
| create_project | pub_key       | {pub_key}       |
| message        | module        | project         |
