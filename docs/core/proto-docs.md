specifying the general details about a buy, sell, or swap order.


| Field | Type | Label | Description |
| ----- | ---- | ----- | ----------- |

| cancelled | [bool](#bool) |  |  |
| cancel_reason | [string](#string) |  |  |

| Field | Type | Label | Description |
